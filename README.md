# How to install 💫

⭐ __PIP:__ ⭐

```code
pip install shaksz
```

__GitHub(ssh)__:
```code
pip install git+ssh://git@github.com/vermavarun/shaksz
```

__GitHub(https)__:
```code
pip install git+https://github.com/vermavarun/shaksz.git
```

__GitHub(Token)__:
```code
pip install git+https://<TOKEN>@github.com/vermavarun/shaksz.git
```
